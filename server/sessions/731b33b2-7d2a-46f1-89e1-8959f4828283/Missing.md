# Missing Points
_학생이 누락하거나 불충분하게 설명한 항목들_

- [ ] 알고리즘이 계산 문제를 해결하는 방법이며, 정확하고 효율적인 설계 및 분석의 중요성을 이해한다
- [ ] 알고리즘의 정확성과 실행 시간 보증이 합성 가능성, 확장성 및 더 나은 알고리즘 설계에 왜 필수적인지 설명할 수 있다
- [ ] 카라추바 및 슈트라센 곱셈 알고리즘을 통해 분할 정복 기법이 기존 방법보다 효율적일 수 있으며, 재귀 관계가 실행 시간 분석에 사용됨을 이해한다
- [ ] 알고리즘 정확성의 중요성 및 합성 가능성, 확장성과의 연관성 설명 누락
